Submerged Installation Guide

Step 1: Download the Mod

Click the button below to download Submerged:

Download Submerged

Step 2: Open Steam Console

Click the button below to open Steam console:

Open Steam Console

Step 3: Install Among Us Version

Enter this command in Steam console (click to copy):

download_depot 945360 945361 7098949171778272067

Step 4: Install the Mod

  1. Open your Among Us game folder (usually C:\Program Files (x86)\Steam\steamapps\content\app_945360)
  2. Copy files from the archive to this folder, preserving the folder structure
  3. Confirm file replacement if needed

Step 5: Launch the Game

Launch the game normally through Steam depot. The mod should load automatically.

If you encounter issues, make sure BepInEx is installed correctly and all mod files are in the right folders.
